Learn & Review: Module 1

1. Regular and Irregular Plurals

Nouns change form to show quantity. Learn the rules:

Rule Pattern Examples
Add -s Most nouns cat → cats, book → books
Add -es s, x, z, ch, sh box → boxes, dish → dishes
Change y → ies Consonant + y baby → babies, city → cities
Add -s (y) Vowel + y boy → boys, key → keys
Change f/fe → ves f or fe ending knife → knives, wife → wives
Irregular Unique changes man → men, child → children, person → people

2. Demonstratives: This, That, These, Those

Use demonstratives to point to or identify things based on distance and number:

SINGULAR (One) PLURAL (Many)
NEAR This (e.g., this pen) These (e.g., these pens)
FAR That (e.g., that pen) Those (e.g., those pens)

✓ Correct: This apple is red. These apples are fresh.

✓ Correct: That mountain is far. Those mountains are tall.

✗ Incorrect: This apples are red. That apple are fresh.

3. Imperatives: Commands & Instructions

Imperatives give orders or instructions. Use the base form of the verb (no subject needed):

✓ Affirmative Imperatives:

  • • Open the door.
  • • Sit down.
  • • Read the book carefully.
  • • Listen to me.

✗ Negative Imperatives (Don't + verb):

  • • Don't open the door.
  • • Don't sit there.
  • • Don't run in the hallway.
  • • Don't eat with your mouth full.

💡 Key Rule: Affirmative imperatives use the base verb (run, walk, eat). Negative imperatives use "Don't" + base verb (don't run, don't walk, don't eat).

Learn & Review: Module 2

1. Present Continuous (am/is/are + -ing)

Use: For actions happening right now, at this moment.

Structure: Subject + be (am/is/are) + V-ing

Examples:

  • • I am reading a book.
  • • She is watching TV.
  • • They are playing football.
  • • The baby is sleeping now.

Time Expressions:

  • • now
  • • right now
  • • at this moment
  • • at the moment

V-ing Spelling Rules:

Rule Example
Just add -ing work → working, talk → talking, read → reading
Drop silent 'e', add -ing make → making, write → writing, take → taking
Double final consonant, add -ing swim → swimming, run → running, sit → sitting
ie → y, add -ing lie → lying, die → dying, tie → tying

2. Present Simple vs. Present Continuous

Choose the right tense based on whether the action is happening now or is a habit/general fact:

Present Simple Present Continuous
Habits & routines Actions happening now
General facts Temporary situations
Permanent situations Planned future events
I play tennis. I am playing tennis.
She works here. She is working now.

3. Critical: Stative Verbs (No -ing!)

Some verbs describe states (conditions) rather than actions. These verbs CANNOT be used with -ing:

Common Stative Verbs:

like, love, hate, want, need, know, understand, remember, forget, believe, think, have, be, seem, appear, look (+ adjective), sound, taste, smell, own, possess

✗ WRONG (with -ing):

  • • I am liking this song.
  • • She is wanting coffee.
  • • They are knowing the answer.
  • • He is owning a car.

✓ CORRECT (without -ing):

  • • I like this song.
  • • She wants coffee.
  • • They know the answer.
  • • He owns a car.

💡 Remember: Use Present Simple (not Continuous) with stative verbs, even when talking about "right now."

Learn & Review: Module 3

1. Comparative Adjectives (+ than)

Use: To compare two people, things, or groups. Structure: Adjective + -er + than OR more + Adjective + than

Spelling Rules for One-Syllable Adjectives:

Rule Example
Most: add -er tall → taller, short → shorter, warm → warmer
End in 'e': add -r large → larger, nice → nicer, brave → braver
End in consonant+y: change y → i, add -er happy → happier, easy → easier, busy → busier
Vowel+consonant: double consonant, add -er big → bigger, fat → fatter, hot → hotter

Multi-Syllable Adjectives: Use "more"

  • • beautiful → more beautiful
  • • interesting → more interesting
  • • expensive → more expensive
  • • comfortable → more comfortable

Examples with "than":

  • • This car is faster than that one.
  • • She is more intelligent than her brother.
  • • Coffee is hotter than juice.

2. Superlative Adjectives (the + -est/most)

Use: To indicate the extreme quality of three or more items. Structure: the + Adjective + -est OR the + most + Adjective

One-Syllable Adjectives:

Rule Example
Most: add -est tall → the tallest, short → the shortest, cold → the coldest
End in 'e': add -st large → the largest, nice → the nicest, brave → the bravest
End in consonant+y: change y → i, add -est happy → the happiest, easy → the easiest, pretty → the prettiest
Vowel+consonant: double consonant, add -est big → the biggest, hot → the hottest, wet → the wettest

Multi-Syllable Adjectives: Use "most"

  • • beautiful → the most beautiful
  • • interesting → the most interesting
  • • expensive → the most expensive
  • • comfortable → the most comfortable

Examples:

  • • This is the fastest car I've ever seen.
  • • She is the most intelligent student in the class.
  • • That mountain is the highest in the region.

3. Irregular Comparative & Superlative Forms

Some adjectives have irregular forms that don't follow the standard rules:

Base Comparative Superlative
good better the best
bad worse the worst
far further / farther the furthest / the farthest
little less the least
much / many more the most

Examples:

  • • This pizza is better than that one. (not "gooder")
  • • His score was worse than mine. (not "bader")
  • • This is the best movie I've ever watched.
  • • That was the worst experience of my life.
  • • The city is farther than I thought.
